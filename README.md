# 日本职场日语 Prompt 生成器

这是一个手机友好的 PWA 小工具。

它的目标是：

> 输入中文，选择对象、渠道、目的、礼貌度和长度，然后生成一段可以复制到 ChatGPT 的高质量 Prompt，让 ChatGPT 帮你改写成自然、得体、符合日本职场语感的日语。

这个版本是 **无 API Key 版**。

也就是说：

- 不需要 OpenAI API Key
- 不会调用 OpenAI API
- 不会产生 API 使用费用
- 可以部署到 GitHub Pages
- 可以添加到手机主屏幕
- 更接近 App 使用体验

---

## 在线使用

如果已经部署到 GitHub Pages，请点击：

[打开日本职场日语 Prompt 生成器](https://你的用户名.github.io/你的仓库名/)

## 这个工具适合做什么？

适合帮助中文母语者在日本职场中整理更自然的日语表达，例如：

- 给上司发 Slack / Teams
- 给同僚发简短消息
- 给团队整体做通知
- 写公司内部邮件
- 写客户邮件
- 做晨会报告
- 做进度汇报
- 请求别人帮忙
- 表达自己想确认
- 表达自己想商量
- 表达任务进度延迟
- 表达体调不良、请假、迟到、早退
- 把直译感较强的日语改得自然一点
